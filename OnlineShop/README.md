## Project OnlineShop using .Net Core API (7) and Angular (16)
```
+ Business domain: e-Commerce application
+ Backend API:
    - REST API, gRPC with .NET (7)
    - Postgresql
    - MongoDb
    - Redis Caching
    - RabbitMQ
    - Real-Time (SignalR)
    - CQRS (Command Query Reposibility Separation)
    - ORM: EF Core, Dapper
    - Event Sourcing
    - ...
+ Frontend:
    - Angular (16)
```
